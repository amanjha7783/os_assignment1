# 🖥️ OS Lab Assignment

This repository contains all **Operating System Lab assignments (A1–A6)** completed using **WSL Ubuntu and VS Code**.

---

## 📌 Assignments Included

### 🔹 Assignment 1 — Linux Basics

* File and directory operations
* Commands: `ls`, `mkdir`, `touch`, `mv`, `cp`, `rm`

### 🔹 Assignment 2 — Shell Scripting

* Bash scripting fundamentals
* Factorial program
* Menu-driven address book
* File renaming automation

### 🔹 Assignment 3 — System Calls

* C programs using system calls
* `open()`, `read()`, `write()`, `close()`
* Tools: `strace`, `gdb`

### 🔹 Assignment 4 — Process Management

* Process commands: `ps`, `pstree`, `kill`, `nice`, `renice`
* Process creation using `fork()`
* Parent-child execution analysis

### 🔹 Assignment 5 — CPU Scheduling

* Algorithms implemented in C:

  * FCFS (First Come First Serve)
  * SJF (Shortest Job First)
  * Round Robin
  * Priority Scheduling
* Performance metrics:

  * Waiting Time
  * Turnaround Time
  * Response Time

### 🔹 Assignment 6 — Memory Management

* Memory Allocation:

  * First Fit
  * Best Fit
  * Worst Fit
* Page Replacement:

  * FIFO
  * LRU
